## 下面的哪个调用是非法的？为什么？哪个调用虽然合法但显然与程序员的初衷不符？为什么？
```cpp
char *init(int ht, int wd = 80, char bckgrnd = ' ');
```
    (a)
```cpp
init()
```
        非法，因为没有传入 ht 的实参。
    (b)
```cpp
init(24, 10)
```
        合法
    (c)
```cpp
init(14, '*');
```
        调用虽然合法但显然与程序员的初衷不符,因为 14 会赋给 ht， '*' 会转换成 int 类型赋给 wd，这显然不符合程序员初衷。
